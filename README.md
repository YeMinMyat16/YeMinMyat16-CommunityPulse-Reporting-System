# CommunityPulse Reporting System

## Overview
CommunityPulse is a community issue reporting system that allows users
to submit, track, and manage complaints through a centralized platform.
The system focuses on reliability, clear workflows, and backend integrity.

## Key Features
- User authentication and authorization
- Complaint submission and status tracking
- SQL-based data storage and retrieval
- Role-based system logic
